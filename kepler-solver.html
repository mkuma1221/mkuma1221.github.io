Kepler’s Problem Solver

Project Overview and Goals

  • Validate and compare two orbital propagation methods: analytical Keplerian propagation and numerical integration under ideal two-body orbital dynamics.
  • Compute the spacecraft’s Classical Orbital Elements (COEs) from an initial Earth-Centered Inertial (ECI) state vector.
  • Apply the Newton–Raphson method to solve Kepler’s Equation and generate a high-accuracy analytical reference solution over five orbital periods.
  • Implement MATLAB’s adaptive ode45 solver to numerically integrate the two-body equations of motion using error tolerances of 10-2, 10-6, and 10-10.
  • Analyze how numerical solver tolerance affects orbital propagation accuracy and long-term solution stability.
  • Compare ECI position and velocity histories between analytical and numerical solutions.
  • Visualize the spacecraft trajectory in three-dimensional space.
  • Evaluate conservation of specific mechanical energy and specific angular momentum to quantify numerical drift and integration accuracy.

Methodology

  • The project began with an initial Earth-Centered Inertial (ECI) position and velocity state vector, which defined the spacecraft’s motion relative to Earth and served as the starting condition for all orbital calculations.
  • Classical Orbital Elements (COEs) were computed from the initial state vectors to describe the orbit in terms of its size, shape, and orientation. These included the semi-major axis, eccentricity, inclination, Right Ascension of the Ascending Node (RAAN), argument of periapsis, and true anomaly.
  • The orbital elements were determined using the spacecraft’s specific orbital energy, angular momentum vector, and eccentricity vector derived directly from the ECI state vectors.
  • A Direction Cosine Matrix (DCM) was constructed to transform coordinates between the ECI frame and the perifocal frame. The perifocal frame aligns directly with the orbit geometry, simplifying orbital propagation calculations and coordinate transformations.
  • Keplerian propagation was performed analytically over five orbital periods by first computing the orbital period and mean motion from the semi-major axis.
  • Kepler’s Equation was solved using the Newton–Raphson iterative method. The key aspect of this approach was selecting an effective initial guess to ensure rapid convergence.
  • The mean anomaly was used as the initial guess for the eccentric anomaly because the two values are generally close for elliptical orbits. This provided a stable starting point for the iteration process and allowed the solution to converge efficiently within the specified tolerance.
  • Once the eccentric anomaly was obtained, it was converted into true anomaly and orbital radius at each time step to generate the spacecraft trajectory throughout the propagation interval.
  • The propagated perifocal position and velocity vectors were transformed back into the ECI frame using the DCM, producing the full three-dimensional orbit in inertial coordinates.
  • Numerical propagation was also performed using MATLAB’s ode45 solver to integrate the two-body equations of motion directly using tolerances of 10-2, 10-6, and 10-10.
  • Analytical and numerical solutions were compared by plotting ECI position histories, velocity histories, and three-dimensional orbital trajectories.
  • Finally, specific orbital energy and specific angular momentum were evaluated throughout the propagation to verify conservation properties and assess numerical integration accuracy.
