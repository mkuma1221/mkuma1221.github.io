Kepler’s Problem Solver

Project Overview and Goals

  • Validate and compare two orbital propagation methods: analytical Keplerian propagation and numerical integration under ideal two-body orbital dynamics.
  • Compute the spacecraft’s Classical Orbital Elements (COEs) from an initial Earth-Centered Inertial (ECI) state vector.
  • Apply the Newton–Raphson method to solve Kepler’s Equation and generate a high-accuracy analytical reference solution over five orbital periods.
  • Implement MATLAB’s adaptive ode45 solver to numerically integrate the two-body equations of motion using error tolerances of 10-2, 10-6, and 10-10.
  • Analyze how numerical solver tolerance affects orbital propagation accuracy and long-term solution stability.
  • Compare ECI position and velocity histories between analytical and numerical solutions.
  • Visualize the spacecraft trajectory in three-dimensional space.
  • Evaluate conservation of specific mechanical energy and specific angular momentum to quantify numerical drift and integration accuracy.

Methodology

  • The project began with an initial Earth-Centered Inertial (ECI) position and velocity state vector at t = 0, which defined the spacecraft’s motion relative to Earth and served as the starting condition for all orbital calculations.
  • Classical Orbital Elements (COEs) were computed from the initial state vectors.
  • A Direction Cosine Matrix (DCM) was constructed to transform between ECI and perifocal frames.
  • Keplerian propagation was performed using mean motion and orbital period relationships.
  • Kepler’s Equation was solved using Newton–Raphson iteration with the mean anomaly as the initial guess.
  • Perifocal state vectors were transformed back into ECI using the DCM.
  • ODE45 was used for numerical propagation with multiple tolerance levels.
