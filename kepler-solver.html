--- --- Kepler Solver Project

Project Overview and Goals

This project implements a numerical solution to Kepler’s Equation using the Newton–Raphson method. It computes orbital parameters from state vectors and validates results using conservation of energy and angular momentum.

Methodology

  • Newton–Raphson iteration for eccentric anomaly
  • Classical Orbital Elements computation
  • ECI ↔ Perifocal transformations using DCM
  • Energy & angular momentum validation

Results

Newton–Raphson vs ODE45 validation
