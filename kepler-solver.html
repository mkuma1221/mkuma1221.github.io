Kepler’s Problem Solver

Project Overview and Goals

  • Validate analytical Kepler propagation vs numerical two-body integration.
  • Compute Classical Orbital Elements (COEs) from initial ECI state vectors.
  • Use Newton–Raphson to solve Kepler’s Equation over 5 orbital periods.
  • Compare ODE45 numerical propagation using different tolerances.
  • Analyze stability using energy and angular momentum conservation.

Methodology

  • Initial ECI state vector is defined at t = 0 and used as the starting condition.
  • Classical Orbital Elements (COEs) are computed from the ECI state vector to describe orbit geometry.
  • Orbital geometry is derived using specific orbital energy, angular momentum, and eccentricity vectors.
  • A Direction Cosine Matrix (DCM) is used to transform between ECI and perifocal frames.
  • Mean motion and orbital period are computed from semi-major axis.
  • Kepler’s Equation is solved using Newton–Raphson iteration.
  • The mean anomaly is used as the initial guess since it is very close to the eccentric anomaly for elliptical orbits, ensuring fast convergence.

Results

  • Analytical Kepler propagation closely matches numerical ODE45 results at tight tolerances.
  • Larger tolerances introduce visible drift in state vectors and orbital invariants.
